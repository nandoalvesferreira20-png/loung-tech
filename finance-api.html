Finance API

API REST para gerenciamento de usuários, autenticação, transações e despesas financeiras.

Stack

Tecnologias Utilizadas

JWT

Autenticação baseada em tokens.

MySQL

Banco de dados relacional.

REST API

Arquitetura moderna baseada em recursos.

Endpoints

Documentação

GET

/users

Retorna todos os usuários cadastrados.

POST

/login

Realiza autenticação e retorna JWT.

GET

/transactions

Lista todas as transações do usuário.

POST

/expenses

Cria uma nova despesa.

DELETE

/expenses/{id}

Remove uma despesa existente.
